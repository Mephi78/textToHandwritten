## Usage

Install [Pillow][pil]  

```bash
$ cd path/to/your/project
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip3 install pillow
```

Paste your text into `MyText.txt` and save the file.  
Run the program `$ python3 maincode.py`.
